# ReversedTicTacToe-miraiPlugin
一个可以跟群友下“反井字棋”的插件。

反井字棋，顾名思义，就是谁先连成三个谁就输的棋类游戏。

命令列表：

| #创建房间          | 创建一个新房间，bot会告知你房间号 |
| ----------------- | :-------------------------------: |
| #进入房间[房间号] | 进入一个房间                      |
| #开始游戏 | 开始下棋 |
| 下\[行数][列数] | 下棋。行数和列数在棋盘上有标记 |
| #联机大厅 | 获取当前群内所有的房间 |
| #退出房间 | 退出一个房间。如果对局中退出将会自动判负 |
